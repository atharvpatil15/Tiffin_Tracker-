# **App Name**: TiffinTrack

## Core Features:

- Daily Tiffin Logging: Users can log their daily tiffin selection (breakfast, lunch, dinner) with pre-defined costs.
- Calendar View: A calendar view displays which days tiffins were taken, with tiffin type indicated on each day.
- Tiffin Tracking: Daily tiffin options can be ticked off to keep track of consumption.
- Multiple User Support: Supports multiple users with individual tiffin tracking and billing.
- Real-time Data Sync: Utilize Firebase Realtime Database to sync data in real time across all users.
- Monthly Bill Generation: Generates a monthly bill based on tiffin consumption with custom start date.
- Custom Monthly Date Range: Users can set a custom start date for their monthly tiffin cycle (e.g., from the 21st of one month to the 21st of the next).

## Style Guidelines:

- Primary color: Warm blue (#468B97) to evoke reliability and calmness.
- Background color: Light off-white (#F0F4F8), providing a clean and modern base.
- Accent color: Muted orange (#D68A36), for highlights and action items, adding a touch of vibrancy without overwhelming.
- Body and headline font: 'PT Sans', a humanist sans-serif with a modern and slightly warm feel suitable for both headlines and body text
- Use simple, outlined icons for tiffin types (breakfast, lunch, dinner) and calendar actions.
- Employ a clean and straightforward layout with a prominent calendar view and clear, actionable buttons. Use Shadcn components.
- Subtle transitions when toggling tiffin options and navigating between months in the calendar.
